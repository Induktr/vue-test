# Руководство Искателя Знаний: "Insight Keeper v2.0"

### 1. Введение: Ваш Цифровой "Второй Мозг"

**Добро пожаловать в "Insight Keeper"!**

Это ваше персональное приложение для организации и управления вашими идеями, задачами и важными цитатами. "Insight Keeper" — это ваш **цифровой "второй мозг"**, спроектированный для **упрощения сложного (Принцип Маргулана)** и повышения вашей **продуктивности**.

### 2. Основные Функции

*   **Создание Постов:**
    *   Создавайте записи трех типов: **"Инсайт"**, **"Задача"**, **"Цитата"**.
    *   Каждый пост может иметь **теги**, **приоритет** и дату создания.
*   **Система Тегирования:**
    *   Добавляйте любое количество тегов к вашим постам.
    *   Все уникальные теги отображаются в панели фильтрации.
*   **Продвинутая Фильтрация:**
    *   Быстро находите посты по поисковому запросу или по выбранным тегам.
*   **Управление Приоритетами:**
    *   Устанавливайте приоритет для каждого поста (1, 2, 3), чтобы визуально выделять наиболее важные.
*   **Пакетные Операции:**
    *   Автоматизируйте рутину: например, удаляйте старые посты, чтобы поддерживать порядок.
*   **Аналитика:**
    *   Получайте быструю статистику по структуре ваших постов.

### 3. Интерфейс Приложения

*   **Список Постов:** Главный экран с карточками ваших записей.
*   **Панель Фильтрации:** Расположена сверху, включает строку поиска и кнопки для выбора тегов.
*   **Кнопки Действий:** Добавление нового поста, запуск пакетных операций.

### 4. Поддержка

Если у вас возникли вопросы, пожалуйста, обратитесь к документации или создайте Issue в репозитории.
