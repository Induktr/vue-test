# Документация Дизайна "Insight Keeper"

### 1. Философия: "Минимализм и Функция"

Дизайн проекта **максимально упрощен**, чтобы не отвлекать от основной цели — анализа кода и логики.

### 2. Цветовая Палитра

| Категория | Цвет (HEX) | Назначение |
|:---|:---|:---|
| **Фон** | `#FFFFFF` | Чистый, простой фон. |
| **Основной Текст** | `#2C3E50` | Высококонтрастный текст. |
| **Кнопка "Добавить"** | `#42B883` (Vue Green) | Основное позитивное действие. |
| **Кнопка "Удалить"** | `#E74C3C` (Red) | Деструктивное действие. |

### 3. Типографика

*   **Шрифт:** Системный sans-serif. **Прагматичный (Принцип 12)** выбор для максимальной производительности.
*   **Иерархия:**
    *   Заголовок (`h1`): `24px`, Bold.
    *   Текст поста (`p`): `16px`, Regular.

### 4. Композиция

*   Простой одноколоночный макет.
*   Используются базовые отступы для разделения элементов.
